# Assets

Esta pasta contém os recursos visuais do Music Player.

## Ícones

- **spotify.ico**: Ícone principal do aplicativo (usado no executável .exe)
- **spotify.png**: Versão PNG do ícone (fonte original)

## Adicionando seu próprio ícone

Para usar um ícone personalizado:

1. Coloque seu arquivo de ícone aqui como `spotify.ico`
2. Ou use uma ferramenta online para converter PNG para ICO:
   - [ICO Convert](https://icoconvert.com/)
   - [Convertio](https://convertio.co/png-ico/)
3. Execute `python build_exe.py` para gerar o executável com o novo ícone

### Recomendações

- **Formato**: ICO (Windows Icon)
- **Tamanhos**: Múltiplas resoluções (16x16, 32x32, 48x48, 256x256)
- **Fundo**: Transparente (para melhor aparência)
